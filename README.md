## Установка зависимостей

В корне проекта выполните:

```bash
npm install
```

или коротко:

```bash
npm i
```

## Запуск в режиме разработки

```bash
npm run dev
```

После запуска Vite покажет адрес (обычно `http://localhost:5173`), откройте его в браузере.

## Сборка проекта

```bash
npm run build
```
